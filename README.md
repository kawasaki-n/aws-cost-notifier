# AWS Cost Notifier

当月の AWS の利用料金を LINE に通知します。

# Description

月初から当日までの AWS 利用料金を LINE Notify に通知します。  
トータルの金額と、サービスごとの金額の両方を確認することができます。  
毎日日本時間 18 時に通知を行います。
